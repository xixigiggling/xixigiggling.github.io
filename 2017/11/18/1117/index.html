vue怎么让v-for出来的项目，当点击其中一项时，输出该项的内容？

遇到的大致情形是：输入搜索词，弹出搜索项，点击搜索项时，得到此搜索项的内容(标题)，并使得父组件的文章立即切换到相应标题的文章

实践:

<!-- 子组件 -->
<div class="item" v-for="title in titles" v-on:click="emitMyEvent($event)">{{ title }}</div>

emitMyEvent(event) {
  var i=0;
  for( i=0;i<5;i++){
    if(event.currentTarget.textContent === this.titles[i]) {
      break;
    }
  }
  this.$emit('my-event',this.titles[i], this.authors[i], this.contents[i]);
}

<!-- 父组件 -->
<searchArticle @my-event="getMyEvent"></searchArticle>  

getMyEvent (title,author,content) {
  this.content = content;
  this.title = title;
  this.author = author;
}

用emitMyEvent($event)来将当前触发点击事件的元素信息发送给父元素

用event.currentTarget.textContent来获取当前元素的内容，再将其与title数组比较，确定是哪个元素(i)，并由此将对应的标题，作者，内容，发送给父元素

父元素用@my-event来接收传参，这样就实现了数据的流动
