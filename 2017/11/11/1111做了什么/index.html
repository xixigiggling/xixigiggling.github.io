在display:none 到display:block之间怎么加动态效果？

那个是从：opacity: 1; 到 opacity: 0; 跟display没关系，display表示展现方式，透明度才能确定渐变程度。

使用方法：

元素需要变化的属性和transition属性在一起:

start {
    opacity: 0;
    transition: opacity .5s ease-in-out;
    ...    
}

再在hover时改变该属性即可:

start:hover {
    opacity:0.6;
}

前提： 鼠标悬停在某元素上出现X的图标(遮罩层)，再次点击可以将该元素的text清空

待解… 有点不稳定，因为不确定是不是，有时可以有时又不行

想要直接拖放一个元素的值到另一个元素上，却发现由于遮罩层的影响，会让增添的文本节点增添到遮罩层里面…
function drop(ev) {
  ev.preventDefault();
  let data=ev.dataTransfer.getData("Text");
  let text = document.createTextNode(data);
  if(ev.target.nodeName == "I") {  
    //加一个判断语句即可解决... I是<i>图标元素
    ev.target.parentNode.parentNode.appendChild(text);
  }else 
    ev.target.parentNode.appendChild(text);
}
新知识点
  1. event.target可以获得触发该事件的节点
  2. ev.target.nodeName 节点名称
  3. childNodes属性返回节点的子节点集合，注意是第一层子节点，且包括文本和换行
  4. node.removeChild(node) 注意是删除node
