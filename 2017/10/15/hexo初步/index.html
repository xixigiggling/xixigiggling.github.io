Hexo搭博客初步

发表于 2017-10-15 | 分类于 Hexo建站

初始化

创建文件夹 

hexo init
git clone https://github.com/iissnan/hexo-theme-nextthemes/next（next为主题名字）

_config.yml的配置

注意里面有很多链接可以参考，具体不表

命令解释

hexo clean
hexo g
hexo s
hexo d
  • 清除public，当 source 文件夹中的部分资源更改过之后，特别是对 文件进行了删除或者路径的改变之后，需要执行这个命令，然后重新 编译。编译，一般部署上去的时候都需要编译一下，编译后，会出现一 个 public 文件夹，将所有的md文件编译成html文件
  • 开启本地服务，
  • 部署

本地浏览博客

在hexo文件夹目录下：

hexo g 
hexo s

localhost：4000即可

写文章

HEXO\source_posts文件下，新建.md文件就可以写文章

hexo new " 文章标题 "

会新建好模版md

部署

_config.yml进行配置deploy即可

发布

发布到Github ：
hexo clean && hexo g && hexo d
