Hexo踩过的一些坑

发表于 2017-10-30 | 分类于 Hexo建站
hexo g报错a multiline key may not be an imp licit key

很可能是没加空格所致，但这次改来改去还是不行

title: spring boot 学习
categories: 
-后端 
-spring boot 
tags: spring boot

如上所示，每个冒号后面都留了空格啊，后来发现就是那个分类的地方，-后端其中的连字符与文字之间也要空格…

hexo不显示markdown语法的标题？

###jpa中的一些配置
页面直接显示出#号…

原因是#号后面也要加空格

hexo在_post里添加的md文件给了tags: xx的分类，点击页面的分类选项还是显示不出来？
  1. hexo new page tags之后会在source文件夹生成一个tags的文件夹里面有index.md文件

  2. 最最重要的是：

    <del>title: tags</del>
date: 2017-10-29 22:11:37
type: tags

    要在默认的index.md那个文件里删除title字段，然后加上type:tags即可

  3. 最后看看有无变化，不然需要hexo clean来重新生成
